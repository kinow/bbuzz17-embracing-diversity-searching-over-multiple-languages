What is Multi-Lingual Search ?

Initial Conditions
Lucene Index containing text from multiple languages
User Query
“schokolade am stiel berlin alexanderplatz” (German)
Search Corpus
Corpora consisting of Italian, German, English documents indexed by Lucene
Search Results
Retrieve query results from translated into German from all languages

Why Multi-Lingual Search ?

  • Most online tech content is in English
  • Good number of non-English speaking users prefer using native language
  • Most search queries are composed in English
  • Preferable to retrieve search results in Native language
  • Embrace Diversity

What is Machine Translation ?

Generate Translations from Statistical Models trained on Bilingual Corpora.

Translation happens per a probability distribution p(e/f)

E = string in the target language (English)

F = string in the source language (Spanish)

e = argmax p(e/f) = argmax p(f/e) * p(e)
